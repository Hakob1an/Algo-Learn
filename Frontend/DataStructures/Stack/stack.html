Stack

A **Stack** is a **linear data structure** that follows the **LIFO (Last In, First Out) principle**. It means that the **last element added is the first one removed**—just like a stack of plates in a cafeteria.

Real-World Examples of a Stack

  • 🔙 **Browser History** – Clicking "Back" removes the most recent page first.
  • 📝 **Undo Feature in Editors** – The most recent change is undone first.
  • 📞 **Call Stack in Programming** – Function calls are processed in a LIFO manner.

Basic Stack Operations

  • ✅ **Push** – Add an element to the **top** of the stack.
  • ❌ **Pop** – Remove the **top** element.
  • 🔍 **Peek (Top)** – View the **top** element without removing it.
  • 📏 **Size** – Get the **number of elements** in the stack.

Step-by-Step Example

📌 Step 1: Start with an empty stack

[]

➕ Step 2: Push elements onto the stack

Adding 5, 8, and 3.

Push(5)  →  [5]
Push(8)  →  [5, 8]
Push(3)  →  [5, 8, 3]  (Top of stack)

❌ Step 3: Pop an element

The topmost element 3 is removed.

Pop() → [5, 8]

Stack Pseudocode

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if len(self.items) == 0:
            return "Stack is empty"
        return self.items.pop()

    def peek(self):
        return self.items[-1] if self.items else "Stack is empty"

    def size(self):
        return len(self.items)

Stack Time Complexity

  • 📌 **Push** – O(1)
  • ❌ **Pop** – O(1)
  • 🔍 **Peek** – O(1)
  • 📏 **Size** – O(1)
